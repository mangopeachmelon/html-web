⚽ 유럽 5대 축구 리그 정보 웹사이트 (European Top 5 Leagues)

축구를 좋아하는 대학생들이 모여 만든, 유럽 축구 입문자를 위한 가이드 웹사이트

유럽을 대표하는 5대 축구 리그(프리미어리그, 라리가, 세리에 A, 분데스리가, 리그 앙)의 특징과 주요 팀, 핵심 선수를 소개하는 반응형 웹사이트입니다. 복잡한 정보 검색 없이도 한눈에 팀의 정보를 파악할 수 있도록 직관적인 카드 디자인으로 제작했습니다.

프로젝트 개요
*프로젝트 기간:** 2025년 12월 1일(월) ~ 2025년 12월 13일(토) (약 2주)
*참여 인원: 3명 (팀 프로젝트)
*프로젝트 목표
    1. HTML/CSS의 기초를 활용하여 웹사이트의 전체적인 구조를 잡는다.
    2. 팀원들과 협업하여 30개 이상의 페이지를 통일감 있게 제작한다.
    3. 반응형 웹을 적용하여 모바일에서도 깨지지 않는 화면을 구현한다.

사용 기술 (Tech Stack)

#Frontend
*HTML5: 웹 표준을 지키며 시맨틱 태그(`header`, `main`, `section`, `footer` 등)를 적극적으로 사용했습니다.
*CSS3:
    - `Flexbox`와 `Grid`를 사용하여 깔끔한 카드 레이아웃을 잡았습니다.
    - `Glassmorphism` (유리 질감) 효과를 헤더에 적용하여 트렌디한 느낌을 주었습니다.
    - `@media` 쿼리를 사용하여 스마트폰 화면에서도 잘 보이는 반응형 웹을 구현했습니다.
- JavaScript (Vanilla JS):**
    - 햄버거 메뉴 버튼을 눌렀을 때 부드럽게 나오는 사이드바를 구현했습니다.
    - 페이지 이동 시 `Fade-in/out` 효과를 넣어 뚝 끊기는 느낌 없이 자연스럽게 넘어가도록 만들었습니다.

✨ 주요 기능 및 특징

1. 직관적인 네비게이션
* 어떤 페이지에 있더라도 상단 헤더와 사이드바를 통해 다른 리그나 메인 화면으로 쉽게 이동할 수 있습니다.
* 모바일 환경에서는 햄버거 메뉴(`☰`)를 통해 공간을 효율적으로 활용했습니다.

2. 통일감 있는 디자인 (Design System)
* 폰트: 가독성이 좋은 `Pretendard` 폰트를 사용하여 깔끔함을 더했습니다.
* 카드 UI: 모든 팀 리스트를 카드 형태로 만들어 마우스를 올리면(Hover) 떠오르는 효과를 주어 클릭하고 싶게 만들었습니다.
* 상세 페이지: 30개 팀 모두 [팀 소개 - 주요 선수 - 네비게이션]의 통일된 구조를 적용하여 사용자가 정보를 쉽게 찾을 수 있게 했습니다.

3. 디테일한 시각 효과
* 배경에 은은한 도트(점) 패턴을 넣어 밋밋하지 않게 채웠습니다.
* 선수 프로필 사진을 원형으로 가공하고 그림자를 주어 입체감을 살렸습니다.
* 리그별, 팀별 특징을 잘 나타내는 고화질 이미지를 선별하여 사용했습니다.

# 💡 프로젝트 후기 & 배운 점
* 협업의 중요성: 파일 경로(`../img/`)를 맞추는 과정에서 팀원 간의 소통이 얼마나 중요한지 깨달았습니다. 초반에 파일명 규칙을 정하는 것이 효율적이라는 것을 배웠습니다.
* CSS의 재사용성: `style.css` 하나만 잘 만들어두면 모든 페이지의 디자인을 한 번에 바꿀 수 있다는 점이 매우 편리했습니다.
* 반응형 웹: PC 화면뿐만 아니라 모바일 화면까지 고려하여 레이아웃을 짜는 것이 생각보다 까다로웠지만, 완성 후 뿌듯함을 느꼈습니다.
